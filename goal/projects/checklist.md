# Checklist for Login Page Testing

## Page & UI
- [ ] Login page loads successfully
- [ ] Email input field is visible
- [ ] Password input field is visible
- [ ] Login button is visible
- [ ] Login button is clickable

## Email Field Validation
- [ ] Email field accepts valid email format
- [ ] Email field shows error for invalid email format
- [ ] Email field shows error when left empty
- [ ] Email field accepts uppercase letters
- [ ] Email field clears after page refresh

## Password Field Validation
- [ ] Password field accepts valid password
- [ ] Password field shows error when left empty
- [ ] Password characters are masked while typing
- [ ] Password field clears after page refresh

## Login Functionality
- [ ] User can log in with valid credentials
- [ ] User cannot log in with invalid password
- [ ] User cannot log in with empty fields
- [ ] Error message is displayed for incorrect credentials
- [ ] Error message text is clear and understandable

## Buttons & Behavior
- [ ] Login button responds on first click
- [ ] Login button behavior is correct with empty fields
- [ ] Login button behavior is correct with invalid data

## General
- [ ] Validation messages are displayed correctly
- [ ] No unexpected behavior after page refresh
