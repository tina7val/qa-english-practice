# QA English Phrases & Mini Dictionary

## Common QA Terms
- Test case
- Bug report
- Preconditions
- Steps to reproduce
- Actual result
- Expected result
- Severity
- Priority
- Status

## Useful QA Sentences
- The issue occurs when...
- User is redirected to...
- Validation message is displayed
- The button is not responsive
- The field accepts invalid input

## Severity Levels
- Low — cosmetic issue
- Medium — affects usability
- High — blocks main functionality
- Critical — system does not work
